# Shoe Tracker

Shoe Tracker is a shoe-tracking application for the Android operating system.

## Project Requirements
* Minimum SDK: 21
* Compile SDK: 30
* Build Tools: 30.0.2
* Java Version: 1.8

## Installation
Clone the Shoe Tracker repository:

`$ https://github.com/kelocen/shoe-tracker.git`

## More Information
This is a proof-of-competency app that was created to fulfill the requirements of an educational assignment. The purpose of this application is to demonstrate a design that includes:
* Single Activity Architecture
* Data Binding
* Navigation Graph
* ViewModel
* Live Data
